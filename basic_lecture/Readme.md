#### section 23 : for 반복문

for문은 조건에 맞춰서 반복문을 실행시킬 수 있다.


#### section 24 : for in 반복문

```
for(var propertyName in obj){
    console.log("\t", propertyName, ": ", obj[propertyName]);
}
```
위에 있는 코드처럼 obj배열을 돌아서 변수로 받습니다.
